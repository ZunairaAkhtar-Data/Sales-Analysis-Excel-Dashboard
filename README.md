# Sales Analysis Excel Dashboard

## Description
This project is an Excel-based sales analysis dashboard designed to evaluate sales performance across different regions and sales executives. It helps identify sales trends, target achievement, and performance gaps through clear and visual analytics.

## Key Analysis & Insights
- Region-wise and city-wise sales comparison
- Sales executive performance evaluation
- Target achieved vs target gap analysis
- Interactive charts, KPIs, and visual summaries

## Tools Used
- Microsoft Excel
